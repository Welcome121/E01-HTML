Manipulação de Arquivos em C

  • Porque manipular aquivos?

    •       Os arquivos permitem o armazenamento de informações. Nesse sentido os dados captados durante a execução de um programa podem ser gravados e, logo, utilizadas em uma compilação futura, assim, permitindo a criação de projetos mais robustos, como o sistema de estoque de uma pizzaria.

          Na linguagem C, utilizamos dois tipos de arquivos para armanezar dados.

    1. Arquivo texto: guarda um array de carracteres, ou seja, uma string.
      Vantagens: fácil output de informações, pode ser interpretado idependente do código que o criou.
      Desvantagens: o processo de leitura e escrita de informações é lento em comparação com a outra opção de arquivo.

    2. Arquio binário: responsável por armazenar uma sequência de bits.
      Vantagens: alto desempenho: em relação tanto a leitura e escrita de dados, quanto a economia de memória.
      Desvantagens: por ser uma sequência de bits, códigos distintos podem interpretar esses arquivos de maneiras diferentes. Assim, apresenta certa depêndencia do código que o criou

  • Variável FILE*

    •       Em C, utilizamos um ponteiro do tipo FILE (presenta na biblioteca stdio.h) para armazenar o endereço de um registro.

    FILE* arquivo;

          Após a criação da variável, podemos efetivamente abrir um arquivo, seja ele criado no próprio código ou pré existente na máquia. Para tal, utilizamos a função fopen() que recebe como parâmetro o local que o arquivo se encontra/deve colocado e o modo de abertura.
    Exemplo: O código abaixo abre o recém criado receitas.txt

    FILE* arquivo;
    arquivo = fopen(receitas.txt, w);

          Veja abaixo outros modos de abertura de arquivo:

          Depois de aberto o arquivo pode receber ou transmitir informações.
    Gravar/escrever: função fprintf(). Primeiro parâmetro indica o arquivo onde seram gravados os dados. Já o segundo são os dados em si. Ler: função fscanf(). Primeiro parâmetro indica onde os dados seram lidos e o segundo controla a forma de extração das informações.

    float preco;
    FILE* arquivo1, arquivo2;
    arquivo1 = fopen(receitas.txt, w);
    arquivo2 = fopen(cardapio.txt, r);
    (void)fprintf(arquivo1, "Bolo de cenoura...");
    (void)fscanf(arquivo2, "%d", &preco);

    Por fim, quando acabados de utilizar nossos arquivos, devemos fechá-los.

    float preco;
    FILE* arquivo1, arquivo2;
    arquivo1 = fopen(receitas.txt, w);
    arquivo2 = fopen(cardapio.txt, r);
    (void)fprintf(arquivo1, "Bolo de cenoura...");
    (void)fscanf(arquivo2, "%d", &preco);
    fclose(arquivo1);
    fclose(arquivo2);
